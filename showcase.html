Here's a list of things people are doing for their 2020 Side Project Summer

Side Project Summer is back for its second year after a wide range of publishing people built some amazing technical projects in 2019.

Add your contribution to this showcase – either your 2019 or 2020 project – by opening a pull request! Follow the instructions on cloning a website on GitHub to take a local copy of this website. Edit the showcase.html file, copying and pasting one of the examples already there to add your own project. Feel free to add screenshots to the images folder and reference them using the <img> HTML tag. Then open a pull request on GitHub, and the General Products maintainers will review it, suggest any changes, and ultimately merge it in.
